## Running `Spider-Man`
### Running `online`
- Go to https://rededge967.github.io/Spider-Man/
### Running `offline`
- Download `source.code.zip` from [here](https://github.com/RedEdge967/Spider-Man/releases)
- Extract the folder
- Run `index.html` file in the browser
